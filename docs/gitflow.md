# CESIZen Gitflow

Ce dépôt suit un flux simple et lisible, adapte a GitHub et a une soutenance.

## Branches

- `main` : version stable, deployable.
- `dev` : branche d'integration pour tout le travail en cours.
- `feature/<sujet>` : une fonctionnalite par branche.
- `fix/<sujet>` : correction ciblee sur `dev`.
- `hotfix/<sujet>` : correction urgente a partir de `main`.

## Convention de commits

Format recommande : `type(scope): message court`
